# ✈️ TripGenius — AI Travel Planner

TripGenius is an intelligent AI-powered travel planning platform that creates personalized travel itineraries based on destination, budget, duration, and travel preferences.

Built using Google Gemini AI, Firebase, Google Maps API, and Next.js, the platform helps users plan smarter and faster trips with AI-generated recommendations, budget analysis, and interactive travel insights.
